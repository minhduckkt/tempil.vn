Product features

How does Tempilaq® Advanced work?

  • ✓ Apply to surface — when rated temperature is reached, the liquid changes colour irreversibly
  • ✓ Quick-dry formula — adheres to horizontal and vertical surfaces without running
  • ✓ Excellent ±1% accuracy — permanent inspection record for QC and audits
  • ✓ Wide temperature range 300°F to 1500°F — covers welding and heat-treatment applications
  • ✓ Use only Green Label Thinner (24597) for dilution — ensures accuracy
  • ✓ Genuine USA import — C/O (Certificate of Origin) and Mill Certificate included
