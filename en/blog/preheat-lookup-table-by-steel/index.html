Preheat Temperature Lookup Table by Steel Type — AWS D1.1, ASME B31.3, API 1104

Published 17 March 2025 · Fast Group Engineering · 10 min read

This page consolidates minimum preheat temperatures from the three most widely applied international welding standards in Vietnam's industrial sector: AWS D1.1 (structural steel welding), ASME B31.3 (process piping), and API 1104 (oil and gas pipeline welding). Data is presented bilingual English with corresponding Tempilstik® part numbers for rapid field look-up at the work site.

How to use this page: Identify the applicable standard from your WPS → find the steel category or P-Number → read the minimum preheat temperature → select the matching Tempilstik® part number. If your WPS specifies a value that differs from this table, always use the WPS value — this table is a technical reference document, not a project specification.

Bilingual Technical Glossary

Minimum preheat temperature
Nhiệt độ preheat tối thiểu — the floor that must be reached before arc strike
Nominal thickness / Base metal thickness
Chiều dày danh nghĩa — governs preheat category in AWS D1.1
Carbon Equivalent (CE)
Chỉ số tương đương carbon — calculated from MTC/MTR values; determines AWS D1.1 steel category
Heat Affected Zone (HAZ)
Vùng ảnh hưởng nhiệt — the metallurgically altered zone adjacent to the weld fusion line
Maximum interpass temperature
Nhiệt độ liên hàn tối đa — the ceiling that must not be exceeded between weld passes
Post Weld Heat Treatment (PWHT)
Xử lý nhiệt sau hàn — separate thermal cycle applied after welding is complete

Table 1 — AWS D1.1: Structural Steel Welding

Applied to: bridges, industrial buildings, offshore structures, module topsides, jackets.
Apply to: structural steel welding — bridges, buildings, offshore structures, jackets.

Carbon Equivalent formula (CE) — AWS D1.1:

CE = C + Mn/6 + (Cr+Mo+V)/5 + (Ni+Cu)/15

All element values (C, Mn, Cr, Mo, V, Ni, Cu) are taken from the Mill Test Certificate (MTC) / Material Test Report (MTR) supplied with the steel lot. CE values must come from the actual material documentation — not assumed from the grade alone.

Table 2 — ASME B31.3: Process Piping

Applied to: process piping in refineries, chemical plants, power generation, and industrial gas facilities. Classified by P-Number — not by Carbon Equivalent.

What is a P-Number? ASME assigns a P-Number to weld base materials based on chemical composition and mechanical properties. The P-Number for a specific pipe is found in the approved WPS or on the Mill Certificate supplied with the pipe.

⚠ Note for Vietnam offshore and field conditions: For oil and gas pipeline welding in Vietnam, WPS preheat temperatures are typically 20–30°C above the API 1104 table minimums to account for high humidity, sea wind, wet pipe surfaces, and offshore site exposure. Always apply the WPS project value — do not use the table minimum directly in offshore applications.
