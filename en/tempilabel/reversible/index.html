Product features

Why choose Tempilabel® Reversible?

  • LCD technology — changes color when the temperature is reached and reverts as it cools, for real-time monitoring
  • 7 temperature levels shown simultaneously — know exactly which level the equipment has reached
  • Reusable — no need to replace after each reading, ideal for long-term continuous monitoring
  • Different from Series 21/4 (Irreversible) — use it when you need the current temperature, not only the peak temperature record
  • Genuine import — U.S. certificate of origin (C/O), C/Q and full VAT invoice
