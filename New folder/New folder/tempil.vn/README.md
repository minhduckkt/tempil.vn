# tempil.vn

Website bán hàng chính hãng Tempil® tại Việt Nam.

## Cấu trúc
- `index.html` — Trang chủ
- `tempilstik/` — Bảng part numbers Tempilstik® (series 28xxx)
- `tempilaq/` — Tempilaq® Advanced  
- `tempilabel/` — Tempilabel®
- `bao-gia/` — Form báo giá
- `contact.html` — Liên hệ
- `assets/` — CSS, logo, ảnh sản phẩm
